07. 브라우저 객체 : window 객체 메서드 : window.blur() : 페이지 포커스 벗어났을 때 설정

요소가 포커스를 잃는 순간(blur), 요소가 포커스를 잃는 것은 대개 '데이터 입력이 완료되었다’는 것을 의미하기 때문에 포커싱이
해제되는 순간엔 데이터를 체크하거나 입력된 데이터를 저장하기 위해 서버에 요청을 보내는 등의 코드를 실행할 수 있다.


    <style>
        .invalid { border-color: red; }
        #error { color: red }
      </style>
      
      이메일: <input type="email" id="input">
      
      <div id="error"></div>
      
      <script>
      input.onblur = function() {
        if (!input.value.includes('@')) { // @ 유무를 이용해 유효한 이메일 주소가 아닌지 체크
          input.classList.add('invalid');
          error.innerHTML = '올바른 이메일 주소를 입력하세요.'
        }
      };
      
      input.onfocus = function() {
        if (this.classList.contains('invalid')) {
          // 사용자가 새로운 값을 입력하려고 하므로 에러 메시지를 지움
          this.classList.remove('invalid');
          error.innerHTML = "";
        }
      };
      </script>

16. 브라우저 객체 : Window 객체 메서드 : Window.moveBy() : 브라우저 위치를 상대적으로 이동

지정해 준 값을 여러번 누르면 누른만큼 이동한다.

{
    <button onclick="openWin()">Open "myWindow"</button>
    <button onclick="moveWin()">Move "myWindow"</button>
    
    <script>
    let myWindow;
    
    function openWin() {
      myWindow = window.open("", "", "width=400, height=400");
    }
    
    function moveWin() {
      myWindow.moveBy(300, 250);
    }
    </script>
}

}

32. 브라우저 객체 : location 객체 메서드 : location.replace() : 위치 이동

location 객체에 속해있는 메소드로 페이지를 이동하는데 사용한다.
assign()메소드나 href속성을 교체하는 것과 replace()가 다른점은 replace()를 사용하고나면 현재의 페이지는 session History에 저장되지 않는다.
이 말은 back 버튼을 눌러도 다시 이 전 페이지로 이동할 수 없다는 뜻이다.

{
    location.replace("이동할 페이지 주소");
}

33. 브라우저 객체 : location 객체 메서드 : location.toString() : 주소 문자열 반환

전체 URL을 담고 있는 USVString을 리턴한다. HTMLHyperlinkElementUtils.href 와 동일한 기능이지만 값을 수정하기 위해 사용할 수는 없다.

{
    // <a id="myAnchor" href="/en-US/docs/Location/toString"> 요소가 문서에 있다고 가정을 했을때
        const anchor = document.getElementById("myAnchor");
        const result = anchor.toString(); // Returns: 'https://developer.mozilla.org/en-US/docs/Location/toString' // 반환 : 'https://developer.mozilla.org/en-US/docs/Location/toString'
}

34. 브라우저 객체 : location 객체 속성 : location.href : 주소

전체 URL을 담고있는 USVString을 리턴하는 stringifier이다. 값이 바뀌면 연관된 document는 새로운 페이지로 이동한다.(주로 쓰임)

{
    // 문서에 <a id="myAnchor" href="https://developer.mozilla.org/en-US/Location/href"> 요소가 있다고 가정했을때
        var anchor = document.getElementById("myAnchor");
        var result = anchor.href; // 반환 : 'https://developer.mozilla.org/en-US/Location/href'
}

35. 브라우저 객체 : location 객체 속성 : location.host : 호스트 이름

host를 담고 있는 USVString이다. (:와 URL의 포트번호를 포함한다.)

{
    var anchor = document.createElement("a");

    anchor.href = "https://developer.mozilla.org/en-US/Location.host"
    anchor.host == "developer.mozilla.org"
    
    anchor.href = "https://developer.mozilla.org:443/en-US/Location.host"
    anchor.host == "developer.mozilla.org"
    // 443이 스키마의 기본 포트이므로 포트 번호는 포함되지 않습니다.
    
    anchor.href = "https://developer.mozilla.org:4097/en-US/Location.host"
    anchor.host == "developer.mozilla.org:4097"
}

36. 브라우저 객체 : location 객체 속성 : location.hostname : 호스트 이름

URL의 도메인을 담고 있는 USVString이다.

{
    var anchor = document.createElement("a");
    anchor.href = "https://developer.mozilla.org:4097/en-US/docs/Location.hostname";
    
    console.log(anchor.hostname);  // developer.mozilla.org
}

37. 브라우저 객체 : location 객체 속성 : location.port : 포트 번호

URL의 포트 번호를 담고 있는 USVString이다.

{
    // 문서에 <a id="myAnchor" href="https://developer.mozilla.org:443/en-US/docs/Location.port"> 요소가 있어야함.
        var anchor = document.getElementByID("myAnchor");
        var result = anchor.port; // Returns:'443'

38. 브라우저 객체 : location 객체 속성 : location.pathname : 디렉토리 경로

최초 / 뒤에 나오는 path들을 담고 있는 USVString이다. (query string이나 fragment는 포함하지 않는다.)

{

40. 브라우저 객체 : location 객체 속성 : location.search : 요청 매개변수

? 를 포함하여 그 뒤에 나오는 URL의 파라미터나 querystring을 담은 USVString이다.
모던 브라우저들은 querystring으로부터 parameter들을 쉽게 파싱할 수 있도록 URLSearchParams나 URL.searchParams등을 제공해준다.

{
    // <a id="myAnchor" href="/en-US/docs/Location.search?q=123"> 요소가 문서에 있도록한다.
        var anchor = document.getElementById("myAnchor");
