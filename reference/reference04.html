상수

상수(constant)는 간단하게는 변수의 반댓말로 常(항상 상)자를 사용해 변하지 않고 고정된 값을 담을 때 사용한다.
변하지 않는 값, 이걸을 제외하면 나머지는 변수와 개념은 동일하다.

상수를 만드는 방법: 상수 선언

상수 선언 예제

const 상수이름 = 상수값;
console.log(상수이름);

const로 선언한 변수를 '상수(constant)'라고 부릅니다.

상수 선언 오류 1

const 상수이름;
상수이름 = 상수값; // Uncaught SyntaxError: Missing initializer in const declaration

처럼 선언 할 수 있을 것 같지만, 상수는 앞서 설명한 것 처럼 선언 이후에 값을 대입할 경우
Uncaught SyntaxError: Missing initializer in const declaration 이라는 에러를 만나게 된다.
따라서 상수는 반드시 선언과 동시에 값을 대입(초기화)해줘야 한다.

상수 선언 오류 2

const 상수이름 = 상수값1;
const 상수이름 = 상수값2; // Uncaught SyntaxError: Missing initializer in const declaration

당연히 재선언 불가능이다.

상수 선언 오류 3

let 상수이름 = 1;
const 상수이름 = 1; // Uncaught SyntaxError: Missing initializer in const declaration

변수 이름과 겹치는 것도 불가능하다.

상수 이름 규칙

const PI = 3.14;
const MY_NUMBER = 1234; // 대문자 + 스네이크 표기법(snake_case)

const pi = 3.14;
const myNumber = 1234; // 카멜 표기법(camel case)

일반적으로 상수를 대문자 + 스네이크 표기법을 사용한다.
하지만 const를 굳이 상수로 구분하지 않고, 암묵적으로 변수라고 함께 묶어 변수를 선언하는 방식으로
이해하고 네이밍 규칙도 카멜 표기법으로 사용한다.

상수를 사용하는 이유

무분별한 자율은 경우에 따라 심각한 문제를 야기할 수 있고,
적당한 제한과 규칙은 경우에 따라 안전이나 안정을 가져다준다.
따라서 상수의 특성은 코드를 상대적으로 엄격하게 사용할 수 있는 장점이 생긴다.
